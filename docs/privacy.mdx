# 秋水游戏隐私保护指引

更新发布日期：2025年8月4日

概要

欢迎您使用秋水游戏，徐州秋水伊人信息技术有限公司（以下简称"我们"）承诺将严格按照相关法律法规的要求处理并保护您的个人信息！除 [《秋水游戏许可及服务协议》](/docs/contract) 外，我们还将通过《秋水游戏隐私保护指引》（"本指引"）向您进一步细化说明您在使用秋水游戏时我们收集、使用、存储和共享个人信息的情况，以及您所享有的相关权利等事宜。本指引是 [《秋水游戏许可及服务协议》](/docs/contract)的组成部分。 您可以通过阅读完整版《秋水游戏隐私保护指引》，了解个人信息类型与用途的对应关系等更加详尽的个人信息处理规则。  
【特别提示】本政策适用于我们提供的所有秋水游戏服务以及子公司游戏。当您使用我们任何单项服务时，您同意隐私政策仅代表您已了解应用提供的功能，以及功能运行所需的必要个人信息，并不代表您已同意我们可以收集非必要个人信息，非必要个人信息会根据您使用过程中的授权情况单独征求您的同意。当您同意接受本政策以及我们在该单项服务中设置的特定隐私信息类政策条款（以下称"特定条款"）的保护，在此情况下特定条款与本政策条款同时对您产生效力。如特定条款与本政策条款存在同类条款的不一致约定，则在特定条款约束范围内应以特定条款为准。如我们提供的某一单项服务不适用本政策的，该服务中会以适当方式明示排除适用本政策。  
在使用秋水游戏前，请您务必仔细阅读并透彻理解本政策，特别是以粗体/粗体下划线标识的条款，您应重点阅读。您同意隐私政策仅代表您已了解应用提供的功能，以及功能运行所需的必要个人信息，并不代表您已同意我们可以收集非必要个人信息，非必要个人信息会根据您使用过程中的授权情况单独征求您的同意。如果您拒绝我们收集非必要个人信息，并不会影响您使用基本业务功能。相应设备权限并不会默认开启，当涉及重要或敏感的设备权限时，我们会在您使用到相应业务功能时，另行弹窗再次征得您的同意后开启；权限开启后，您还可以随时通过设备设置关闭权限；你不同意开启权限，将不会影响其他非相关业务功能的正常使用。针对个人敏感信息，我们会单独征得您的同意后进行处理。如果您不同意本政策的内容，将导致秋水游戏无法正常运行，或者无法达到我们拟达到的服务效果，您应立即停止访问/使用秋水游戏。  
为满足防沉迷等相关法律法规政策或相关主管部门的要求，我们可能在国家有关网络游戏防沉迷政策规定的基础上，实施更为严格的防沉迷措施。在您发起支付、出现可疑登录或异常消费行为等特定场景时，我们的部分游戏或针对部分用户(以具体游戏策略为准)可能会不时启用人脸识别进行验证，旨在为了更精准地识别并保护未成年人，最大限度防止未成年人冒用他人身份信息进行游戏。上述人脸识别验证主要是将您的真实面部信息和身份信息与公安权威数据平台数据源进行比对，如您提供的信息与公安威数据平台数据源比对结果一致，即成功通过认证，如对比结果不符或您拒绝验证，您当次操作将受到限制，可能无法继续游戏或在一定时间内影响部分功能的使用。您的人脸信息为敏感个人信息，我们将采取严格的保密措施，且仅用于与公安数据平台进行比对，我们不会存储您人脸信息。具体请查看本指引第十部分。
